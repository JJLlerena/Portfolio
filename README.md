# Portfolio Website

Personal portfolio showcasing web development projects and technical skills.

## Tech Stack

- React
- React Router
- TailwindCSS styled components
- Deployed on Vercel

## Features

- Single-page navigation with smooth scrolling
- Project preview panels with live demos
- Responsive design
- Section-based routing

## Setup
```bash
npm install
npm run dev
```

Runs on `localhost:3000`

## Build
```bash
npm run build
```
